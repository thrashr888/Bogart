Opricot is an open-source GUI and scripting environment for MongoDB. It contains a Javascript driver that works natively in the browser through an AJAX interface.

Command editor

Enter any Javascript to the command editor window. Built-in functions and objects allow you to manipulate databases from your script. Use print() to show output on this result screen. Enter query parameters and other arrays as native Javascript (JSON syntax). You can often dig deeper into the results by clicking on the interactive command responses.
